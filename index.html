ⓘ 合成原理：本工具使用浏览器内置 Web Speech API (TTS) 进行真实语音合成，通过角色音色参数 + 情感预设调节 pitch/rate/volume 实现情感表达。批量生成时通过 getDisplayMedia + MediaRecorder 捕获每段 TTS 音频，编码为 OGG/WebM Opus 文件，供游戏引擎导入。
